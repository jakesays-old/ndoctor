# Install nDoctor with the installer #

https://lh3.googleusercontent.com/-6JHf-3xgiZM/UNHwlMqdyJI/AAAAAAAAAQ0/QgIXvE0CY4g/s240/1.PNG

  * After downloading the installer [HERE](http://code.google.com/p/ndoctor/downloads/list), double click on the .msi file

![https://lh3.googleusercontent.com/-nI6EzbkIXs4/UNHwlN7lrNI/AAAAAAAAAQ8/DHelJezAKPw/s499/2.png](https://lh3.googleusercontent.com/-nI6EzbkIXs4/UNHwlN7lrNI/AAAAAAAAAQ8/DHelJezAKPw/s499/2.png)

  * Click on NEXT

![https://lh3.googleusercontent.com/-3_xvo9CKvAo/UNHwlNbNwvI/AAAAAAAAAQ4/jgVd7pGBfx0/s499/3.png](https://lh3.googleusercontent.com/-3_xvo9CKvAo/UNHwlNbNwvI/AAAAAAAAAQ4/jgVd7pGBfx0/s499/3.png)

  * Read the [GNU GPL v3](http://www.gnu.org/licenses/gpl.html) accept it and click on NEXT

![https://lh5.googleusercontent.com/-rbukZ863sXY/UNHwlkAFGcI/AAAAAAAAARA/vlLtbnrA1H4/s499/4.png](https://lh5.googleusercontent.com/-rbukZ863sXY/UNHwlkAFGcI/AAAAAAAAARA/vlLtbnrA1H4/s499/4.png)

  * By default, nDoctor is installed in Program Files but you can change the destination directory. When it's done, click on NEXT

![https://lh6.googleusercontent.com/-0qt8BuojWfU/UNHwl8spXfI/AAAAAAAAARE/xwrSKcIPDhE/s499/5.png](https://lh6.googleusercontent.com/-0qt8BuojWfU/UNHwl8spXfI/AAAAAAAAARE/xwrSKcIPDhE/s499/5.png)

  * Just click on INSTALL

https://lh6.googleusercontent.com/-kiXotvoqyak/UNH0vf2rT_I/AAAAAAAAASM/dA5f1Jfuav4/s500/Capture.JPG

  * The UAC ([User Account Control](http://windows.microsoft.com/en-US/windows7/products/features/user-account-control)) will ask you to accept the installation. You have to click on YES

![https://lh6.googleusercontent.com/-eQPkNTfZOqw/UNHwmOOk-eI/AAAAAAAAARI/xEThWpSv50M/s499/6.png](https://lh6.googleusercontent.com/-eQPkNTfZOqw/UNHwmOOk-eI/AAAAAAAAARI/xEThWpSv50M/s499/6.png)

  * Click on FINISH

![https://lh5.googleusercontent.com/-DLXGCUAM2kM/UNHwmjfdH4I/AAAAAAAAARU/KvwxhQbXCYw/s240/7.png](https://lh5.googleusercontent.com/-DLXGCUAM2kM/UNHwmjfdH4I/AAAAAAAAARU/KvwxhQbXCYw/s240/7.png)

  * When the installation is done, you will see a shortcut on the Desktop and in the Start Menu
